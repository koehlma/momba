Quickstart¶

If you would like to use the model for your own research, here is how you get started. To use the model from within Python, we first import the necessary packages:

import random

from momba import engine

from racetrack import model, tracks, svg

The racetrack package allows constructing models based on scenario descriptions. In the case of Racetrack, a scenario description specifies the track and other parameters such as the tank size or the underground (see Scenario). Here, we use the Barto-Big track:

tracks.BARTO_BIG

We construct a scenario by randomly selecting a start cell of that track:

start_cell = random.choice(list(tracks.BARTO_BIG.start_cells))
scenario = model.Scenario(tracks.BARTO_BIG, start_cell, underground=model.Underground.SAND)

As you can see, this gave us an automaton Network.

We can now export the network to JANI, run a model checker on it, or explore it with Momba’s state exploration engine—just to mention a few possibilities. To explore the model with Momba’s state space exploration engine, we construct an Explorer:

explorer = engine.Explorer.new_discrete_time(mdp)
